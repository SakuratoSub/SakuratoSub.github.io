发布规范

发表于 2021-05-03 | 分类于 Fansub

Sakurato作品发布格式标准

新番组

单集发布

单集新番发布直接对作品文件本身进行制种，无需嵌套上级文件夹

合集发布

字幕整理

当合集修正版已完成压制之后，需将最终版字幕文件使用文本编辑器工具（包括但不限于Notepad++、VSCode）打开检查是否包含[Aegisub Project Garbage]字段信息
通常这里会包含打轴的时候选定的音视频流、工作进度等信息，整理过程中需将该段信息全部删除并保存
需要删除的部分↓

删除完成之后把字幕文件按照命名规范中的格式命名，全选所有字幕文件后使用7zip压缩工具压缩
直接将所有字幕文件添加进压缩包根路径，无需在上层嵌套文件夹

字体整理

依次将最终版字幕文件拖入Aegisub工具中，点击左上角文件→字体收集器→检查已使用的字体→开始，查看当前计算机是否已安装全部字体
若无误则会有绿色字体提示提示所有字体已找到。
若缺少字体则需记下所有缺失的字体询问时轴索要

当确认完当前所有字幕中的全部字体都已安装后，再依次将字幕文件拖入Aegisub工具中，点击左上角文件→字体收集器→复制字体至文件夹→选择路径→开始

最终在所指的的文件夹内找到当前所有字幕里的全部字体,全选所有字体文件，创建一个7z解压缩包并压缩

字幕和字体的压缩参数：

  • 压缩格式：7z
  • 压缩等级：极限压缩
  • 压缩方法：LZMA2
  • 字典大小：1024MB
  • 单词大小：256
  • 固实数据大小：固实

合集新番在修订完成之后按照命名规范中的格式命名，且需附带上字幕、字体文件的压缩包一同放进文件夹里进行制种。
